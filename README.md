# Signal

Signal is a contemplative idle game prototype where you rebuild a lost communication lattice between distant civilizations. Manage bandwidth, cultivate data packets, and deploy relay stations as you stretch the range of your network. Unlock exotic frequencies when you contact new cultures, overlap signals for bursts of production, and reboot the universe to retain long-term signal efficiency.

## Getting started

Open `index.html` in a browser to play. No build step or server is required.

## Core loop

- **Ping a Probe** to manually generate bandwidth while accumulating trace data.
- Spend bandwidth and data to **Expand Range**, **Deploy Relay Stations**, and **Optimize Routing**. These upgrades increase passive bandwidth and data generation.
- Reaching specific range thresholds contacts new civilizations, granting exotic frequencies that reshape production math.
- When at least two frequencies are unlocked, **Overlap Signals** to trigger a temporary resonance that dramatically boosts resource flow.
- After rebuilding enough of the lattice, **Reboot the Universe** to begin anew with higher signal efficiency that carries across runs.

This foundation will grow with additional content, balancing, and Steam integration in future iterations.
